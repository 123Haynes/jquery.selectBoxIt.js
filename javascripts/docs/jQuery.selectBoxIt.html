Immediately-Invoked Function Expression (IIFE) Ben Alman Blog Post that locally passes in jQuery, the window object, the document object, and an undefined variable.  The jQuery, window and document objects are passed in locally, to improve performance, since javascript first searches for a variable match within the local variables set before searching the global variables set.  All of the global variables are passed in locally to be minifier friendly. undefined can be passed in locally, because it is not a reserved word in JavaScript
